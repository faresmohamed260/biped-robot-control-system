# Biped Robot Brand Guide

## Identity

The current identity is for the **Biped Robot** project. The visual direction should feel:

- precise
- capable
- field-ready
- software-native

The visual language mixes mechanical geometry with a polished control-room look. The mark is intentionally compact so it works as both a robotics emblem and a launcher icon.

## Core assets

- `logo-horizontal.svg`: primary logo for README banners, splash screens, and presentations
- `logo-mark.svg`: standalone emblem for square placements
- `icon-app.svg`: simplified app icon for launcher surfaces and desktop shortcuts
- `banner-repo.svg`: wide project banner for GitHub or docs headers
- `social-card.svg`: announcement and preview-card asset
- `preview.html`: local browser preview for the full branding pack

## Color system

- `#0E1116` Graphite Core
- `#131922` Panel Shadow
- `#F46A2F` Utility Orange
- `#39D0C7` Control Teal
- `#FFF6EF` Warm Signal
- `#9BA7B8` Muted Interface

## Usage notes

- Prefer dark surfaces when using the full-color logo.
- Use `logo-mark.svg` for square avatars and compact app placements.
- Use `icon-app.svg` when the mark needs cleaner edges at small sizes.
- Keep enough padding around the logo so the claw silhouette remains readable.
- If you need a monochrome version later, derive it from the SVG sources rather than rasterizing first.

## Typography

The current SVG pack uses broadly available system-safe fonts for portability:

- `Segoe UI`
- `Arial`
- generic `sans-serif`

For future product pages or presentation decks, a good upgrade path would be:

- headline: `Space Grotesk` or `Sora`
- UI / body: `Inter`

## Creative direction

- Orange carries actuation, movement, and mechanical power.
- Teal carries telemetry, alignment, and software control.
- Graphite anchors the pack in a tool-like, engineering-driven tone.
